# Device Service Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete device-service that subscribes to MQTT, persists device state in PostgreSQL, writes sensor data to InfluxDB, owns and executes scheduled MQTT commands, broadcasts live state via WebSocket, and exposes REST APIs for device control and schedule CRUD.

**Architecture:** Spring Boot 4.0.5 service using Eclipse Paho for MQTT, JPA for PostgreSQL, influxdb-client-java for InfluxDB writes, STOMP/WebSocket for live broadcast, and OAuth2 Resource Server for JWT validation via auth-service JWKS. Two Flyway-managed tables: `devices` and `schedules`.

**Tech Stack:** Java 25, Spring Boot 4.0.5, Eclipse Paho MQTT 1.2.5, influxdb-client-java 7.2.0, springdoc-openapi 2.7.0, Testcontainers 1.20.4, PostgreSQL 17, InfluxDB 2, Mosquitto 2.

**Spec:** `docs/superpowers/specs/2026-04-06-device-service-design.md`

**Reference codebase:** `../homelab-auth-service/` — follow same conventions for entities, exceptions, controller tests, Dockerfile, YAML config.

**Key SB 4.0 patterns (from auth-service):**
- Jackson 3: `tools.jackson.databind.ObjectMapper`
- WebMvcTest: `org.springframework.boot.webmvc.test.autoconfigure.WebMvcTest`
- MockitoBean: `org.springframework.test.context.bean.override.mockito.MockitoBean` (not `@MockBean`)
- MockMvc: requires `spring-boot-webmvc-test` dependency
- YAML config (not .properties)
- Entity auditing: `@EnableJpaAuditing` + `@CreatedDate`/`@LastModifiedDate` (preferred over `@PrePersist`/`@PreUpdate`)
- Auth-service JWT: custom `role` claim (value: `"USER"` or `"ADMIN"`), NOT in `scope`/`scp` — requires custom `JwtAuthenticationConverter`

**Known limitations (accepted for single-instance homelab):**
- In-memory scheduler (`ThreadPoolTaskScheduler`) — no distributed locking. If multiple replicas deployed, duplicate MQTT commands will fire. Acceptable for single-instance; add ShedLock if scaling.
- Synchronous MQTT client (Eclipse Paho `MqttClient`) — adequate for ~2 devices publishing every 30s. Switch to `MqttAsyncClient` if throughput becomes an issue.
- WebSocket unauthenticated — intentional design: read-only broadcast of non-sensitive device state. Document as accepted risk.

## Task 3: Security config (OAuth2 Resource Server)

**Files:**
- Create: `src/main/java/ch/furchert/homelab/device/config/SecurityConfig.java`

Note: device-service uses OAuth2 Resource Server (validates JWTs from auth-service via JWKS), NOT a custom filter like auth-service. Auth-service JWT has a custom `role` claim (value: `"USER"` or `"ADMIN"`) — requires a custom `JwtAuthenticationConverter` to map to Spring Security authorities.

- [ ] **Step 1: Create SecurityConfig**

Create `src/main/java/ch/furchert/homelab/device/config/SecurityConfig.java`:

```java
package ch.furchert.homelab.device.config;

import lombok.RequiredArgsConstructor;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.core.convert.converter.Converter;
import org.springframework.http.HttpMethod;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configuration.EnableWebSecurity;
import org.springframework.security.config.annotation.web.configurers.AbstractHttpConfigurer;
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.core.GrantedAuthority;
import org.springframework.security.core.authority.SimpleGrantedAuthority;
import org.springframework.security.oauth2.jwt.Jwt;
import org.springframework.security.oauth2.server.resource.authentication.JwtAuthenticationConverter;
import org.springframework.security.web.SecurityFilterChain;
import tools.jackson.databind.ObjectMapper;

import java.util.Collection;
import java.util.List;

@Configuration
@EnableWebSecurity
@RequiredArgsConstructor
public class SecurityConfig {

    private final ObjectMapper objectMapper;

    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
                .csrf(AbstractHttpConfigurer::disable)
                .sessionManagement(s -> s.sessionCreationPolicy(SessionCreationPolicy.STATELESS))
                .authorizeHttpRequests(auth -> auth
                        .requestMatchers("/ws/**").permitAll()
                        .requestMatchers("/actuator/health", "/actuator/info").permitAll()
                        .requestMatchers("/api-docs/**", "/swagger-ui/**", "/swagger-ui.html").permitAll()
                        .requestMatchers(HttpMethod.POST, "/schedules").hasRole("ADMIN")
                        .requestMatchers(HttpMethod.PUT, "/schedules/**").hasRole("ADMIN")
                        .requestMatchers(HttpMethod.DELETE, "/schedules/**").hasRole("ADMIN")
                        .anyRequest().authenticated()
                )
                .oauth2ResourceServer(oauth2 -> oauth2
                        .jwt(jwt -> jwt.jwtAuthenticationConverter(jwtAuthenticationConverter()))
                )
                .exceptionHandling(ex -> ex
                        .authenticationEntryPoint((req, res, e) -> {
                            res.setContentType("application/json");
                            res.setStatus(401);
                            res.getWriter().write(objectMapper.writeValueAsString(
                                    java.util.Map.of("status", 401, "error", "Unauthorized", "timestamp", java.time.Instant.now().toString())));
                        })
                        .accessDeniedHandler((req, res, e) -> {
                            res.setContentType("application/json");
                            res.setStatus(403);
                            res.getWriter().write(objectMapper.writeValueAsString(
                                    java.util.Map.of("status", 403, "error", "Forbidden", "timestamp", java.time.Instant.now().toString())));
                        })
                );

        return http.build();
    }

    @Bean
    public JwtAuthenticationConverter jwtAuthenticationConverter() {
        JwtAuthenticationConverter converter = new JwtAuthenticationConverter();
        converter.setJwtGrantedAuthoritiesConverter(new RoleClaimConverter());
        return converter;
    }

    /**
     * Auth-service JWT has a custom "role" claim with value "USER" or "ADMIN".
     * Maps to Spring Security authority "ROLE_USER" or "ROLE_ADMIN".
     */
    private static class RoleClaimConverter implements Converter<Jwt, Collection<GrantedAuthority>> {
        @Override
        public Collection<GrantedAuthority> convert(Jwt jwt) {
            String role = jwt.getClaimAsString("role");
            if (role == null || role.isBlank()) {
                return List.of();
            }
            return List.of(new SimpleGrantedAuthority("ROLE_" + role));
        }
    }
}
```

```java
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public ResponseEntity<Map<String, Object>> handleValidation(MethodArgumentNotValidException ex) {
        Map<String, String> fieldErrors = ex.getBindingResult().getFieldErrors().stream()
                .collect(Collectors.toMap(FieldError::getField, FieldError::getDefaultMessage, (a, b) -> a));
        return ResponseEntity.badRequest().body(errorBody(HttpStatus.BAD_REQUEST, "Validation failed", fieldErrors));
    }

    @ExceptionHandler(ResourceNotFoundException.class)
    public ResponseEntity<Map<String, Object>> handleNotFound(ResourceNotFoundException ex) {
        return ResponseEntity.status(HttpStatus.NOT_FOUND)
                .body(errorBody(HttpStatus.NOT_FOUND, ex.getMessage(), null));
    }

    @ExceptionHandler(IllegalArgumentException.class)
    public ResponseEntity<Map<String, Object>> handleIllegalArgument(IllegalArgumentException ex) {
        return ResponseEntity.badRequest().body(errorBody(HttpStatus.BAD_REQUEST, ex.getMessage(), null));
    }

    @ExceptionHandler(Exception.class)
    public ResponseEntity<Map<String, Object>> handleGeneric(Exception ex) {
        log.error("Unhandled exception", ex);
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body(errorBody(HttpStatus.INTERNAL_SERVER_ERROR, "Internal server error", null));
    }

    private Map<String, Object> errorBody(HttpStatus status, String message, Object details) {
        return details != null
                ? Map.of("status", status.value(), "error", message, "timestamp", Instant.now().toString(), "details", details)
                : Map.of("status", status.value(), "error", message, "timestamp", Instant.now().toString());
```

```java
import ch.furchert.homelab.device.config.MqttProperties;
import jakarta.annotation.PostConstruct;
import jakarta.annotation.PreDestroy;
import lombok.RequiredArgsConstructor;

import org.springframework.scheduling.annotation.Scheduled;
import org.springframework.stereotype.Service;

@Service
@Slf4j
@RequiredArgsConstructor
public class MqttClientService implements MqttCallbackExtended {

    private static final int MAX_INITIAL_RETRIES = 10;
    private static final long RETRY_DELAY_MS = 5000;

    private final MqttProperties mqttProperties;
    private final MqttMessageHandler mqttMessageHandler;
    private MqttClient mqttClient;
    private MqttConnectOptions connectOptions;
    private volatile boolean initialConnectionEstablished = false;

    @PostConstruct
    void init() {
        try {
            mqttClient = new MqttClient(mqttProperties.getBrokerUrl(), mqttProperties.getClientId());
            mqttClient.setCallback(this);

            connectOptions = new MqttConnectOptions();
            connectOptions.setAutomaticReconnect(true);
            connectOptions.setCleanSession(true);
            connectOptions.setKeepAliveInterval(30);
            connectOptions.setConnectionTimeout(10);

            if (mqttProperties.getUsername() != null && !mqttProperties.getUsername().isBlank()) {
                connectOptions.setUserName(mqttProperties.getUsername());
                connectOptions.setPassword(mqttProperties.getPassword().toCharArray());
            }

            connectOptions.setWill(
                    mqttProperties.getWillTopic(),
                    mqttProperties.getWillPayload().getBytes(),
                    1,
                    true
            );

            attemptConnect();
        } catch (MqttException e) {
            log.error("Failed to create MQTT client: {}", e.getMessage());
        }
    }

    /**
     * Retry initial connection if it failed during startup.
     * Paho's automaticReconnect only works after the first successful connection.
     */
    @Scheduled(fixedDelay = 10000, initialDelay = 10000)
    void retryConnectionIfNeeded() {
        if (!initialConnectionEstablished && mqttClient != null && !mqttClient.isConnected()) {
            log.info("MQTT not connected, retrying initial connection...");
            attemptConnect();
        }
    }

    private void attemptConnect() {
        try {
            mqttClient.connect(connectOptions);
            initialConnectionEstablished = true;
            log.info("MQTT connected to {}", mqttProperties.getBrokerUrl());
        } catch (MqttException e) {
            log.warn("MQTT connection attempt failed: {}. Will retry.", e.getMessage());
        }
    }

    @PreDestroy
    void disconnect() {
        try {
            if (mqttClient != null && mqttClient.isConnected()) {
                publish(mqttProperties.getWillTopic(), "{\"MqttState\": 0}");
                mqttClient.disconnect();
                log.info("MQTT disconnected");
            }
        } catch (MqttException e) {
            log.error("Error disconnecting MQTT: {}", e.getMessage());
        }
    }

    public void publish(String topic, String payload) {
        try {
            if (mqttClient != null && mqttClient.isConnected()) {
                mqttClient.publish(topic, new MqttMessage(payload.getBytes()));
            } else {
                log.warn("MQTT client not connected, cannot publish to {}", topic);
            }
        } catch (MqttException e) {
            log.error("Failed to publish to {}: {}", topic, e.getMessage());
        }
    }

    @Override
    public void connectComplete(boolean reconnect, String serverURI) {
        log.info("MQTT {} to {}", reconnect ? "reconnected" : "connected", serverURI);
        initialConnectionEstablished = true;
        subscribeToTopics();
        publish(mqttProperties.getWillTopic(), "{\"MqttState\": 1}");
    }

    @Override
    public void connectionLost(Throwable cause) {
        log.warn("MQTT connection lost: {}", cause.getMessage());
    }

    @Override
    public void messageArrived(String topic, MqttMessage message) {
        mqttMessageHandler.handleMessage(topic, new String(message.getPayload()));
    }

    @Override
    public void deliveryComplete(IMqttDeliveryToken token) {
        // no-op
    }

    private void subscribeToTopics() {
        try {
            for (String topic : mqttProperties.getTopics()) {
                mqttClient.subscribe(topic, mqttProperties.getQos());
                log.info("Subscribed to MQTT topic: {}", topic);
            }
        } catch (MqttException e) {
            log.error("Failed to subscribe to topics: {}", e.getMessage());
        }
    }
}
```

```java
package ch.furchert.homelab.device.service;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;
import tools.jackson.databind.JsonNode;
import tools.jackson.databind.ObjectMapper;

import java.util.Optional;

@Component
@Slf4j
@RequiredArgsConstructor
public class MqttMessageParser {

    private final ObjectMapper objectMapper;

    public enum MessageType {
        SENSOR_DATA, MQTT_STATUS, LIGHT, NIGHT_LIGHT, RAIN
    }

    public record ParsedMqttMessage(
            String deviceName,
            MessageType type,
            Double temperature,
            Double humidity,
            Boolean mqttOnline,
            String stateValue
    ) {}

    public Optional<ParsedMqttMessage> parse(String topic, String payload) {
        String[] segments = topic.split("/");
        if (segments.length < 2) {
            return Optional.empty();
        }

        String deviceName = segments[0];
        // Only process terra1, terra2, etc. — not terraGeneral
        if (!deviceName.matches("terra\\d+")) {
            return Optional.empty();
        }

        try {
            JsonNode json = objectMapper.readTree(payload);

            return switch (buildTopicKey(segments)) {
                case "SHT35/data" -> Optional.of(new ParsedMqttMessage(
                        deviceName, MessageType.SENSOR_DATA,
                        json.get("Temperature").asDouble(),
                        json.get("Humidity").asDouble(),
                        null, null
                ));
                case "mqtt/status" -> Optional.of(new ParsedMqttMessage(
                        deviceName, MessageType.MQTT_STATUS,
                        null, null,
                        json.get("MqttState").asInt() == 1,
                        null
                ));
                case "light" -> Optional.of(new ParsedMqttMessage(
                        deviceName, MessageType.LIGHT,
                        null, null, null,
                        json.get("LightState").asInt() == 1 ? "ON" : "OFF"
                ));
                case "nightLight" -> Optional.of(new ParsedMqttMessage(
                        deviceName, MessageType.NIGHT_LIGHT,
                        null, null, null,
                        json.get("NightLightState").asInt() == 1 ? "ON" : "OFF"
                ));
                case "rain" -> Optional.of(new ParsedMqttMessage(
                        deviceName, MessageType.RAIN,
                        null, null, null,
                        json.get("RainState").asInt() == 1 ? "ON" : "OFF"
                ));
                default -> Optional.empty();
            };
        } catch (Exception e) {
            log.warn("Failed to parse MQTT message on topic {}: {}", topic, e.getMessage());
            return Optional.empty();
        }
    }

    private String buildTopicKey(String[] segments) {
        if (segments.length == 2) {
            return segments[1];
        }
        return segments[1] + "/" + segments[2];
    }
}
```

```java
     * Rejects unknown fields rather than guessing key names.
     */
    private static final Map<String, String> FIELD_TO_STATE_KEY = Map.of(
            "light", "LightState",
            "nightLight", "NightLightState",
            "rain", "RainState"
    );

    public List<DeviceDto> getAllDevices() {
        return deviceRepository.findAll().stream().map(DeviceDto::from).toList();
    }

    public DeviceDto getDeviceById(Long id) {
        return deviceRepository.findById(id)
                .map(DeviceDto::from)
                .orElseThrow(() -> new ResourceNotFoundException("Device not found: " + id));
    }

    public void sendControlCommand(Long deviceId, ControlCommandDto command) {
        Device device = deviceRepository.findById(deviceId)
                .orElseThrow(() -> new ResourceNotFoundException("Device not found: " + deviceId));

        String stateKey = FIELD_TO_STATE_KEY.get(command.field());
        if (stateKey == null) {
            throw new IllegalArgumentException("Unknown control field: " + command.field()
                    + ". Valid fields: " + FIELD_TO_STATE_KEY.keySet());
        }

        String topic = device.getName() + "/" + command.field() + "/man";
        String payload = "{\"" + stateKey + "\": " + command.state() + "}";
        mqttClientService.publish(topic, payload);
    }

    @Transactional
    public DeviceDto updateFromMqtt(MqttMessageParser.ParsedMqttMessage msg) {
        Device device = deviceRepository.findByName(msg.deviceName())
                .orElseGet(() -> {
                    Device d = new Device();
                    d.setName(msg.deviceName());
                    return d;
                });

        switch (msg.type()) {
            case SENSOR_DATA -> {
                device.setTemperature(msg.temperature());
                device.setHumidity(msg.humidity());
            }
            case MQTT_STATUS -> device.setMqttOnline(msg.mqttOnline());
            case LIGHT -> device.setLight(msg.stateValue());
            case NIGHT_LIGHT -> device.setNightLight(msg.stateValue());
            case RAIN -> device.setRain(msg.stateValue());
        }

        device.setLastSeen(Instant.now());
        device = deviceRepository.save(device);
        return DeviceDto.from(device);
    }
```

```java
package ch.furchert.homelab.device.service;

import ch.furchert.homelab.device.entity.Schedule;
import ch.furchert.homelab.device.repository.ScheduleRepository;
import jakarta.annotation.PostConstruct;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.scheduling.annotation.Scheduled;
import org.springframework.scheduling.concurrent.ThreadPoolTaskScheduler;
import org.springframework.scheduling.support.CronTrigger;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Map;
import java.util.Set;
import java.util.concurrent.ConcurrentHashMap;
import java.util.concurrent.ScheduledFuture;
import java.util.stream.Collectors;

@Service
@Slf4j
@RequiredArgsConstructor
public class SchedulerService {

    private final ScheduleRepository scheduleRepository;
    private final MqttClientService mqttClientService;
    private final ThreadPoolTaskScheduler taskScheduler;

    private final Map<Long, ScheduledFuture<?>> activeTasks = new ConcurrentHashMap<>();
    private final Map<Long, String> taskHashes = new ConcurrentHashMap<>();

    @PostConstruct
    void init() {
        reload();
    }

    @Scheduled(fixedDelayString = "${app.scheduler.poll-interval}")
    public void pollReload() {
        reload();
    }

    public void reload() {
        log.info("Reloading schedules...");

        List<Schedule> activeSchedules = scheduleRepository.findByActiveTrue();
        Set<Long> activeIds = activeSchedules.stream()
                .map(Schedule::getId)
                .collect(Collectors.toSet());

        // Cancel removed or changed schedules
        activeTasks.keySet().stream()
                .filter(id -> !activeIds.contains(id))
                .forEach(this::cancelTask);

        // Register or update schedules
        for (Schedule schedule : activeSchedules) {
            String hash = computeHash(schedule);
            String existingHash = taskHashes.get(schedule.getId());

            if (hash.equals(existingHash)) {
                continue; // no change
            }

            // Cancel old version if exists
            cancelTask(schedule.getId());

            // Register new version
            try {
                CronTrigger trigger = new CronTrigger(schedule.getCronExpression());
                ScheduledFuture<?> future = taskScheduler.schedule(
                        () -> executeSchedule(schedule),
                        trigger
                );
                activeTasks.put(schedule.getId(), future);
                taskHashes.put(schedule.getId(), hash);
                log.info("Registered schedule: {} (id={})", schedule.getTitle(), schedule.getId());
            } catch (IllegalArgumentException e) {
                log.error("Invalid cron expression for schedule {}: {}", schedule.getId(), e.getMessage());
            }
        }

        log.info("Schedules reloaded: {} active tasks", activeTasks.size());
    }

    private void cancelTask(Long scheduleId) {
        ScheduledFuture<?> future = activeTasks.remove(scheduleId);
        if (future != null) {
            future.cancel(false);
            taskHashes.remove(scheduleId);
            log.info("Cancelled schedule task id={}", scheduleId);
        }
    }

    private void executeSchedule(Schedule schedule) {
        log.info("Executing schedule: {} -> {} with payload {}", schedule.getTitle(), schedule.getMqttTopic(), schedule.getMqttPayload());
        mqttClientService.publish(schedule.getMqttTopic(), schedule.getMqttPayload());
    }

    private String computeHash(Schedule schedule) {
        return schedule.getCronExpression() + "|" + schedule.getMqttTopic() + "|" + schedule.getMqttPayload();
    }
}
```

```java
    ScheduleRepository scheduleRepository;
```

```java
    SchedulerService schedulerService;

    @Test
    void scheduleWithImminentCron_publishesMqttMessage() throws Exception {
        String brokerUrl = "tcp://" + mosquitto.getHost() + ":" + mosquitto.getMappedPort(1883);
        MqttClient subscriber = new MqttClient(brokerUrl, "test-subscriber");
        subscriber.connect();

        CountDownLatch latch = new CountDownLatch(1);
        AtomicReference<String> receivedPayload = new AtomicReference<>();

        subscriber.subscribe("terraGeneral/light/schedule", (topic, msg) -> {
            receivedPayload.set(new String(msg.getPayload()));
            latch.countDown();
        });

        // Create a schedule that fires every second
        Schedule schedule = new Schedule();
        schedule.setTitle("Test schedule");
        schedule.setCronExpression("* * * * * *"); // every second
        schedule.setMqttTopic("terraGeneral/light/schedule");
        schedule.setMqttPayload("{\"LightState\": 1}");
        schedule.setActive(true);
        scheduleRepository.save(schedule);

        schedulerService.reload();

        boolean received = latch.await(5, TimeUnit.SECONDS);
        assertThat(received).isTrue();
        assertThat(receivedPayload.get()).isEqualTo("{\"LightState\": 1}");

        subscriber.disconnect();
    }
}
```
